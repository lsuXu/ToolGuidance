# 时间戳转换工具

一个简单易用的时间戳转换工具，支持以下功能：

## 功能特点

- 支持10位和13位时间戳转换
- 实时显示当前时间和时间戳
- 双向转换（时间戳 ↔ 日期时间）
- 实时输入验证和提示
- 自动填充当前时间

## 使用方法

1. 直接打开 `index.html` 文件即可使用
2. 输入时间戳或选择日期时间进行转换
3. 转换结果会实时显示

## 技术栈

- HTML5
- CSS3
- JavaScript (纯前端实现)
